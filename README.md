
`Mi Empresa Online` simula la gestión de una tienda virtual, permite administrar productos, clientes y el inventario utilizando listas y diccionarios.



## Gestión de Productos
- **Agregar productos**: Registra productos con nombre, precio y cantidad.  
- **Actualizar stock**: Incrementa o disminuye la cantidad automáticamente si ya existe.  
- **Eliminar productos**: Retira productos que ya no se venden.  
- **Consultar inventario**: Visualiza todos los productos.
- **Valor total del inventario**: Calcula cuánto vale el inventario.
### Gestión de Clientes (Bonus)
- **Agregar clientes**: Guarda nombre, email y mantiene un historial de compras.  
- **Ver clientes**: Consulta clientes y cuántas compras han realizado.

### Funciones

- **Agregar productos**  
  - `mi_empresa.agregar_producto(nombre, precio, cantidad)`

- **Ver inventario**  
  - `mi_empresa.ver_inventario()`

- **Buscar producto**  
  - `mi_empresa.buscar_producto(nombre)`

- **Actualizar stock**  
  - `mi_empresa.actualizar_stock(nombre, cantidad)`

- **Eliminar producto**  
  - `mi_empresa.eliminar_producto(nombre)`

- **Calcular valor total del inventario**  
  - `mi_empresa.calcular_valor_inventario()`

- **Agregar clientes**  
  - `mi_empresa.agregar_cliente(nombre, email)`

- **Ver clientes**  
  - `mi_empresa.ver_clientes()`
